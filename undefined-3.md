{% tab title="등급" %}
#### 아기 등급 명령어

* `/창고1` : 1번 가상창고를 엽니다.
* `/모자` : 손에 들고 있는 아이템이나 블록을 머리에 착용합니다.

#### 꼬마 등급 명령어

* `/창고2` : 2번 가상창고를 엽니다.
* `/제작대` : 제작대를 엽니다.
* `/밥` : 배고픔을 채웁니다. 쿨타임과 비용은 없습니다.

#### 의젓 등급 명령어

* `/창고3`, `/창고4` : 3번, 4번 가상창고를 엽니다.
* `/줄기보호` : 호박, 수박의 줄기보호를 `on/off` 합니다.
* `/야간투시` : 야간투시 효과를 `on/off` 합니다.
* `슈퍼점프 (Shift+점프)` : 먼 거리를 도약하여 빠르게 이동합니다.

#### 듬직 등급 명령어

* `/창고5`, `/창고6` : 5번, 6번 가상창고를 엽니다.
* `/발광` : 발광을 `on/off` 하거나 색상을 변경합니다.
* `/셋홈`, `/홈` : 홈을 지정하여 빠르게 이동합니다.

#### 대장 등급 명령어

* `/창고7`, `/창고8` : 7번, 8번 가상창고를 엽니다.
* `/원격요리` : 원격으로 요리 메뉴를 엽니다.
* `/원격무역` : 원격으로 무역 메뉴를 엽니다.
* `/원격상점` : 원격으로 상점 메뉴를 엽니다.
* `/한글닉네임` : 한글닉네임을 변경합니다. 횟수 제한 없이 사용할 수 있습니다.
{% endtab %}
