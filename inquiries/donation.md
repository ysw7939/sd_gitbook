# 💎 후원

## 서버 후원

> 후원 링크 : [https://donate.sudalfarm.com/cash](https://donate.sudalfarm.com/cash)

* 후원 시 **1:1 비율로 캐시가 지급**됩니다.
* 금액에 따라 보너스 캐시가 지급됩니다.
* 입금 확인시 자동 캐시 지급이 처리됩니다.

{% hint style="info" %}
문제 발생시 디스코드로 문의 바랍니다.
{% endhint %}

## 누적 후원 보상

누적 후원 금액이 일정 금액을 넘으면 게임 내에서 `/누적후원보상` 명령어로 보상을 우편으로 수령할 수 있습니다. 각 단계는 **1회만** 수령 가능합니다.

| 누적 후원 금액   | 보상 구성                                                                  |
| ---------- | ---------------------------------------------------------------------- |
| 5만원 이상 | <p>· 캐시쿠폰 10개<br>· [소모품] 플라이 쿠폰(10분) 12개<br>· 망원경 1개</p> |

### 이용약관

* 후원금은 어떠한 경우에도 **환불이 불가능**합니다.
* 후원금은 **다른 계정으로 양도/이전이 불가능**합니다.
* **계좌이체를 통한 후원은 현금영수증 발행이 가능합니다.** (문의 티켓)
