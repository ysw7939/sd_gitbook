# 💎 후원

## 서버 후원

> 후원 링크 : [https://donate.sudalfarm.com/cash](https://donate.sudalfarm.com/cash)

* 후원 시 **1:1 비율로 캐시가 지급**됩니다.
* 금액에 따라 보너스 캐시가 지급됩니다.
* 입금 확인시 자동 캐시 지급이 처리됩니다.

{% hint style="info" %}
문제 발생시 디스코드로 문의 바랍니다.
{% endhint %}

## 누적 후원 보상

누적 후원 금액이 일정 금액을 넘으면 게임 내에서 `/누적후원보상` 명령어로 보상을 우편으로 수령할 수 있습니다. 각 단계는 **1회만** 수령 가능합니다.

| 누적 후원 금액  | 보상 구성                                                                                                                                                                                                                                                                  |
| --------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| 5만원 이상    | <p>· 캐시 5,000<br>· [소모품] 플라이 쿠폰(10분) 12개<br>· 망원경 1개</p>                                                                                                                                                                                                                |
| 10만원 이상   | <p>· 피로도 100 회복권 1개<br>· 치장가챠 1개</p>                                                                                                                                                                                                                                    |
| 25만원 이상   | <p>· 피로도 100 회복권 5개<br>· 일일퀘스트 보상 2배<br>· 엘리베이터 차광유리 10개<br>· 치장가챠 2개</p>                                                                                                                                                                                                |
| 50만원 이상   | <p>· 피로도 100 회복권 10개<br>· 왕관 세트 (Kings 치장 교환권)<br>· 왕의 의자 1개 (들고 <code>/의자</code> 사용)<br>· 유저상점 판매슬롯 5→8개로 확장<br>· 핫타임보상 2배<br>· 커스텀 칭호 제작권(텍스트) 1개<br>· 걷날개</p>                                                                                                          |
| 100만원 이상  | <p>· 피로도 100 회복권 20개<br>· 피로도 50 회복권 20개<br>· 발광포션(검정색) 1개<br>· [한정판] 제작수달펫 1개<br>· 자동줍기권 1개<br>· 자동심기 무제한권 1개<br>· 압축 영구권 1개<br>· 플라이 무제한권 1개<br>· 무역 슬롯 5개 전체 해금 (자동 적용)<br>· 커스텀 칭호 제작권(픽셀아트) 1개<br>· 커스텀 모자 치장 제작권 1개</p>                                                                                                            |

{% hint style="info" %}
**커스텀 칭호 제작권**(텍스트/픽셀아트)은 종이 아이템 형태로 지급되며, **디스코드 문의 티켓**으로 신청해 제작받을 수 있습니다.
{% endhint %}

### 이용약관

* 후원금은 어떠한 경우에도 **환불이 불가능**합니다.
* 후원금은 **다른 계정으로 양도/이전이 불가능**합니다.
* **계좌이체를 통한 후원은 현금영수증 발행이 가능합니다.** (문의 티켓)
