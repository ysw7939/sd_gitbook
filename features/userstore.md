# 🛒 유저상점

유저상점은 플레이어 간 아이템을 사고팔 수 있는 거래소입니다.\
`/유저상점` 또는 `/유상` 명령어로 이용할 수 있습니다.

## 기능

| 기능 | 설명 |
| --- | --- |
| 아이템 등록 | 인벤토리의 아이템을 가격을 정하여 등록 |
| 아이템 검색 | 이름이나 키워드로 아이템 검색 |
| 유저 검색 | 특정 유저가 등록한 아이템 검색 |
| 키워드 등록 | 자주 검색하는 키워드 저장 |
| 시세 확인 | 아이템의 최근 거래 시세 확인 |
| 최근 거래 | 최근 거래 내역 확인 |

## 규칙

* 판매 수수료: **5%**
* 등록 가능 개수: 1인당 최대 **50개**
* 키워드 저장: 1인당 최대 **10개**
* 가격 범위: **1원 ~ 999,999,999원**
* 등록 만료: **7일** 후 자동 만료

{% hint style="warning" %}
시세표에 기재된 아이템은 시세 범위 내에서만 거래해야 합니다. 터무니없는 가격에 거래 시 처벌될 수 있습니다.
{% endhint %}
