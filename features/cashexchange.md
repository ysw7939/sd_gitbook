# 💱 캐시거래소

캐시거래소는 플레이어끼리 **캐시를 머니로 사고팔 수 있는 거래소**입니다.\
`/캐시거래소` 또는 `/캐시거래` 명령어로 이용할 수 있습니다.

***

<figure><img src="../.gitbook/assets/image.png" alt=""><figcaption></figcaption></figure>

## 캐시 구매하기

1. `/캐시거래소` 명령어로 캐시거래소 GUI를 엽니다.
2. 원하는 캐시 거래를 확인합니다.
3. 거래 항목을 좌클릭하면 구매가 완료됩니다.
4. 구매한 캐시는 즉시 보유 캐시에 지급됩니다.

{% hint style="info" %}
구매에는 머니가 사용됩니다. 보유 머니가 부족하면 구매할 수 없습니다.
{% endhint %}

***

## 캐시 판매 등록하기

보유 캐시를 판매하려면 `/캐시거래 <캐시금액> <판매머니>` 명령어를 사용합니다.

**예시:**

* `/캐시거래 100 50000` — 100캐시를 50,000원에 등록
* `/캐시거래 100 100*500` — 100캐시를 50,000원에 등록
* `/캐시거래 500 250000` — 500캐시를 250,000원에 등록

{% hint style="warning" %}
판매 등록을 하면 등록한 캐시는 즉시 차감되어 거래소에 보관됩니다.\
등록 중 오류가 발생하면 차감된 캐시는 자동으로 환불됩니다.
{% endhint %}

***

## 나의 물품 관리

캐시거래소 GUI에서 **나의 물품** 버튼을 클릭하면 내가 등록한 캐시 거래를 확인할 수 있습니다.

판매가 완료된 거래는 나의 물품에서 좌클릭하여 판매머니를 수령할 수 있습니다.\
판매자가 접속 중이면 판매 완료 알림도 받을 수 있습니다.

***

## 검색과 정렬

캐시거래소 GUI에서는 등록된 거래를 편하게 찾을 수 있습니다.

* **판매자 검색** — 유저 이름으로 캐시 거래를 검색합니다.
* **정렬 변경** — 최신순, 낮은가격순, 높은가격순, 비율 낮은순, 비율 높은순으로 볼 수 있습니다.
* **목록 새로고침** — 현재 거래 목록을 다시 불러옵니다.

***

## 명령어

| 명령어                   | 설명                  |
| --------------------- | ------------------- |
| `/캐시거래소`              | 캐시거래소 GUI를 엽니다      |
| `/캐시거래`               | 캐시거래소 GUI를 엽니다      |
| `/캐시거래 <캐시금액> <판매머니>` | 보유 캐시를 판매 등록합니다     |
| `/캐시거래 도움말`           | 캐시거래 명령어 도움말을 확인합니다 |

{% hint style="info" %}
판매머니에는 `100*500` 같은 간단한 계산식을 사용할 수 있습니다.
{% endhint %}

***

## 규칙

* 가격 제한: **1캐시당 최대 500원**
* 등록 가능 개수: 1인당 최대 **5개**
* 캐시금액과 판매머니는 **1 이상**이어야 합니다.
* 본인이 등록한 캐시 거래는 직접 구매할 수 없습니다.
* 구매가 완료된 거래의 판매머니는 **나의 물품**에서 직접 수령해야 합니다.

{% hint style="warning" %}
캐시 거래는 실제 보유 캐시와 머니를 사용하는 거래입니다. 금액을 입력하기 전에 캐시 수량과 판매머니를 꼭 확인해주세요.
{% endhint %}
