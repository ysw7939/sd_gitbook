# 🐾 펫

펫은 플레이어를 따라다니며 주변의 **드롭 아이템을 자동으로 수집**해주는 동반자입니다.

## 사용 방법

* 펫 아이템을 사용하면 펫이 소환됩니다.
* 서버 접속 시 보유한 펫이 자동으로 소환됩니다.

## 펫 종류

다양한 펫이 존재하며, 각 펫마다 외형과 수집 능력이 다릅니다.

* 기본 수집 펫
* 슬리피 수달
* 킹덤 캣
* 펭귄
* 피카츄
* 리퍼
* 그 외 다수

## 기능

* 주변 **5\~7블록** 범위 내의 드롭 아이템을 자동 수집합니다.
* 플레이어를 따라다니며, 멀어지면 자동으로 텔레포트합니다.
* **본인의 드롭 아이템만** 수집합니다.

{% hint style="info" %}
펫은 캐시 상점, 이벤트, 가챠 등을 통해 획득할 수 있습니다.
{% endhint %}
