# 🧸 치장 아이템

`/치장` 또는 `/코스튬` 명령어로 치장 메뉴를 열 수 있습니다.\
스폰의 **옷장 NPC**를 우클릭하면 착용 모습을 미리 확인할 수 있습니다.

{% hint style="info" %}
치장 아이템은 착용 전 **미리보기**(30초)를 통해 확인할 수 있습니다.
{% endhint %}

## 치장 슬롯

| 슬롯  | 부위        | 예시                  |
| --- | --------- | ------------------- |
| 모자  | 머리 위      | 동물 모자, 시즌 헬멧, 후드 등  |
| 등치장 | 등         | 가방·날개·망토            |
| 손치장 | 오프핸드(왼손)  | 무기·인형·도구 모형         |
| 풍선  | 머리 위 풍선   | 캐릭터 위로 둥둥 떠다니는 장식   |
| 움직임 | 이동 이펙트    | 발자국, 꽃잎, 하트 등 이동 트레일 |

## 획득 방법

* **수달 세트 뽑기**: [캐시 상점](cashshop.md)에서 3,900 캐시로 구매하는 가챠 박스
* **랜덤 치장 캡슐**: 진주 줍기 시즌 랭킹·이벤트 보상 등으로 획득
* **시즌 한정 번들**: 봄 캠핑·여름·할로윈·부활절·킹덤 등 시즌별 한정 출시
* **이벤트·후원 보상**: 운영 이벤트나 후원 단계별 보상

{% hint style="info" %}
신규 치장 아이템은 시즌·이벤트 일정에 맞춰 추가됩니다. 보유 아이템은 `/치장`에서 항상 확인할 수 있습니다.
{% endhint %}

## 시즌 한정 컬렉션

시즌·이벤트 기간 동안만 등장하는 한정 치장 컬렉션입니다. 해당 시즌이 끝나면 일반 획득 경로에서 사라지므로, 등장 기간 안에 획득해 두시는 것을 권장합니다.

* 🎃 **할로윈** — 마녀빗자루·박쥐 망토·호박 캔디 등
* 🌸 **봄 캠핑** — 캠핑 모자·파라솔·바구니
* 🦋 **봄 나비** — 나비 더듬이·나비 날개·나비 지팡이
* 🥚 **부활절** — 농부 밀짚모자·계란 바구니·토끼 인형 가방
* ☀️ **여름** — 오토바이 가방·구름 풍선·해바라기 풍선
* 👑 **킹덤** — 왕관·로브·홀
* 💖 **이펙트** — 하트·꽃잎·눈송이 등 이동 트레일

{% hint style="info" %}
한정 컬렉션의 정확한 출시·종료 일정은 디스코드 공지를 참고해주세요.
{% endhint %}

## 펫

`/펫` 명령어로 펫을 소환하거나 관리할 수 있습니다.\
펫은 플레이어를 따라다니며 주변의 **드롭 아이템을 자동으로 수집**하고, 일부 펫은 추가로 **🌙 야간투시 · 💨 이동속도 증가 · 🦘 슈퍼점프** 버프를 제공합니다.

{% hint style="info" %}
펫마다 수집 범위와 제공 버프가 다를 수 있습니다. 자세한 정보는 [펫](../undefined-2/pet.md) 페이지를 참고해주세요.
{% endhint %}
