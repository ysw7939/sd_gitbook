# 🧸 치장 아이템

`/치장` 또는 `/코스튬` 명령어로 치장 메뉴를 열 수 있습니다.

{% hint style="info" %}
치장 아이템은 착용 전 **미리보기**(30초)를 통해 확인할 수 있습니다.
{% endhint %}

## 펫

`/펫` 명령어로 펫을 소환하거나 관리할 수 있습니다.\
자석펫은 플레이어를 따라다니며, 주변의 **드롭 아이템을 자동으로 수집**합니다.

{% hint style="info" %}
펫마다 수집 범위와 속도가 다를 수 있습니다.
{% endhint %}
