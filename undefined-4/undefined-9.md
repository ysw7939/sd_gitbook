### 게임 규칙&#x20;

* `/홀짝 등록 [금액] [홀/짝]` 명령어로 게임을 등록합니다.&#x20;
* 상대방이 `/홀짝`  명령어로 등록된 도박을 클릭한 후 홀 또는 짝을 선택해서 맞추면 배팅한 금액을 얻게 됩니다.&#x20;
* 게임을 등록할때 금액이 차감되며, 게임에서 승리하게 되면 배팅한 금액의 2배를 얻게됩니다.&#x20;

| 명령어                  | 설명                     |
| -------------------- | ---------------------- |
| `/홀짝`                | 도박 페이지를 엽니다.           |
| `/홀짝 등록 <금액> <홀\|짝>` | 도박을 등록합니다.             |
| `/홀짝 취소`             | 본인이 등록한 게임등록을 취소합니다.   |
| `/홀짝 알림`             | 홀짝 알림 ON/OFF 를 변경합니다.  |
