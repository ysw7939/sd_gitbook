# 📜 스탯

## 스탯 콘텐츠 안내

서버의 스탯은 `광산 스탯`과 `농사 스탯`으로 나뉘어 있습니다.\
각 스탯은 레벨을 올려 포인트를 얻고, 포인트와 골드를 사용해 원하는 효과를 강화하는 방식입니다.

### 공통 규칙

* 모든 스킬 최대 레벨은 `Lv.5`입니다.
* 스킬 강화에는 `스킬 포인트 + 골드`가 함께 필요합니다.
* 강화 비용
  * `Lv.1`: 포인트 1 / 5,000골드
  * `Lv.2`: 포인트 2 / 12,000골드
  * `Lv.3`: 포인트 5 / 35,000골드
  * `Lv.4`: 포인트 12 / 120,000골드
  * `Lv.5`: 포인트 25 / 400,000골드
* 채굴은 레벨업마다 `스킬 포인트 +5`
* 농사는 레벨업마다 `스킬 포인트 +1`
