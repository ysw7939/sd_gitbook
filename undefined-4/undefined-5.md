# 🚢 무역

## &#x20; ​무역 컨텐츠 소개&#x20;

* 무역은 특정 아이템을 일반 판매가보다 1\~20% 높은 가격으로 판매할 수 있는 거래 시스템입니다.
* 무역 이용을 위해서는 추천이 필요합니다.
* 무역은 24시간 언제든지 가능하며, 시세는 일정 범위 내에서 변동되므로 적절한 타이밍에 거래하면 더 높은 이익을 얻을 수 있습니다.

{% hint style="info" %}
무역은 매일 오전 3시에 초기화 됩니다.&#x20;
{% endhint %}
