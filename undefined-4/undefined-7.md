# 🔖 퀘스트

* `/퀘스트` 명령어로 일일 퀘스트 항목을 확인할수 있습니다.&#x20;
* 하루에 배정되는 퀘스트는 총 `5개`입니다.

<figure><img src="../.gitbook/assets/image (14).png" alt=""><figcaption></figcaption></figure>

#### **고정 퀘스트**

아래 2개는 매일 무조건 들어가는 고정 퀘스트입니다.

| 퀘스트 키            | 이름      | 목표          |
| ---------------- | ------- | ----------- |
| `vote_recommend` | 서버 추천   | 서버를 1회 추천   |
| `harvest_200`    | 채집 200회 | 채집을 200회 진행 |
