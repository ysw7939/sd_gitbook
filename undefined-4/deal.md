# 🤝 유저 거래

유저 거래는 다른 플레이어와 아이템과 돈을 안전하게 주고받을 수 있는 기능입니다.\
거래 GUI에서 서로 올린 아이템과 금액을 확인한 뒤, 양쪽 모두 확인해야 거래가 완료됩니다.

## 거래 신청

`/거래 신청 <닉네임>` 명령어로 거래를 신청할 수 있습니다.\
상대방은 `/거래 수락` 또는 `/거래 거절`로 거래 요청에 응답할 수 있습니다.

{% hint style="info" %}
유저 거래는 같은 서버에 접속 중인 플레이어끼리 사용할 수 있습니다.\
거래 요청은 **30초** 동안 유지되며, 시간이 지나면 자동으로 만료됩니다.
{% endhint %}

## 거래 방법

1. `/거래 신청 <닉네임>`으로 거래를 신청합니다.
2. 상대방이 `/거래 수락`을 입력하면 거래 GUI가 열립니다.
3. 거래할 아이템을 내 거래 칸에 올립니다.
4. 돈을 함께 거래하려면 돈 슬롯을 클릭해 금액을 입력합니다.
5. 양쪽 모두 내용을 확인한 뒤 **준비 완료**를 누릅니다.
6. 양쪽 모두 **최종 확인**을 누르면 거래가 완료됩니다.

## 돈 거래

거래 GUI의 돈 슬롯을 클릭하면 거래 금액을 입력할 수 있습니다.\
돈 거래에는 **1% 수수료**가 적용되며, 수수료를 제외한 금액이 상대방에게 지급됩니다.

## 안전 장치

* 거래 중 한쪽이 아이템을 변경하면 확인 상태가 초기화됩니다.
* 인벤토리 공간이 부족하면 거래가 완료되지 않습니다.
* 돈이 부족하면 거래가 완료되지 않습니다.
* 거래가 취소되면 올려둔 아이템은 다시 돌려받습니다.

## 거래 차단

원하지 않는 유저의 거래 신청을 차단할 수 있습니다.

| 명령어 | 설명 |
| --- | --- |
| `/거래 차단 <닉네임>` | 해당 유저의 거래 신청 차단 또는 해제 |
| `/거래 차단목록` | 내가 차단한 유저 목록 확인 |

## 명령어

| 명령어 | 설명 |
| --- | --- |
| `/거래 신청 <닉네임>` | 유저에게 거래 신청 |
| `/거래 수락` | 받은 거래 요청 수락 |
| `/거래 거절` | 받은 거래 요청 거절 |
| `/거래 차단 <닉네임>` | 유저 거래 차단 또는 해제 |
| `/거래 차단목록` | 거래 차단 목록 확인 |
