# 🎣 낚시

## 낚시 콘텐츠 안내

스폰 서버의 낚시는 단순히 물고기만 낚는 콘텐츠가 아니라,\
다양한 등급의 어종을 수집하고, 더 좋은 낚싯대로 희귀 어종에 도전하며,\
크기와 가격에 따라 수익을 올릴 수 있는 성장형 콘텐츠입니다.

### 낚시는 어떻게 진행되나요?

1. 물가에서 낚싯대를 던집니다.
2. 입질이 오면 미니게임이 시작됩니다.
3. 미니게임에 성공하면 물고기나 아이템을 획득합니다.
4. 잡은 어종은 크기에 따라 더 높은 가격으로 판매할 수 있습니다.
5. 더 좋은 낚싯대를 사용할수록 상위 등급 어종을 만날 확률이 올라갑니다.



### 어종 등급

낚시에서는 어종이 `C급`, `B급`, `A급`, `S급`으로 나뉩니다.

| 등급 | 설명                                      |
| -- | --------------------------------------- |
| C급 | 가장 쉽게 만날 수 있는 등급입니다. 잡템이나 소형 어종이 포함됩니다. |
| B급 | 가장 자주 등장하는 기본 등급입니다. 낚시 수익의 중심이 됩니다.    |
| A급 | 비교적 희귀한 상위 어종입니다. 가격도 높은 편입니다.          |
| S급 | 매우 희귀한 최고급 어종입니다. 현재는 상어가 해당합니다.        |
