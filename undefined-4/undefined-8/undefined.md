# 📖 인첸트 강화

## 인첸트 강화 소개&#x20;

* 인첸트 강화는 장비의 세부 능력을 끌어올리는 콘텐츠입니다.
* 효율, 내구성, 행운이 무작위로 상승하며, 최대 30강까지 도전할 수 있습니다.
* 수달석과 윤기나는 수달석 중 어떤 재료를 선택하느냐에 따라 성공 확률이 달라집니다.

<figure><img src="../../.gitbook/assets/image (8).png" alt=""><figcaption></figcaption></figure>

#### 인첸트 강화 확률표

| 사용 재료    | 적용 구간     |  성공 |  실패 |
| -------- | --------- | --: | --: |
| 수달석      | 1강 \~ 30강 | 45% | 55% |
| 윤기나는 수달석 | 1강 \~ 30강 | 65% | 35% |

{% hint style="info" %}
* 인첸트 강화는 실패 시 강화 수치가 효율4 행운3  내구성3 으로 초기화됩니다.
* 효율4 행운3  내구성3 이전에는 인첸트가 초기화 되지 않습니다.
* 보호 아이템은 사용 시 먼저 소모되며, 보호 성공 시에만 초기화를 막을 수 있습니다.
{% endhint %}
