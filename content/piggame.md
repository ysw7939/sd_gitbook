# 🐷 도축

<figure><img src="../.gitbook/assets/Video Project (1).gif" alt=""><figcaption></figcaption></figure>

도축은 미니게임 형태로 진행되는 컨텐츠입니다.\
돼지를 잡아 고기를 획득하고, 판매하거나 요리 재료로 사용할 수 있습니다.

## 도축장 이동 방법

`/메뉴` 명령어로 메뉴를 열고 **도축장** 버튼을 클릭하면 도축장으로 이동합니다.

## 진행 방법

1. **돼지 우클릭** — 필드에 있는 돼지를 우클릭하면 미니게임이 시작됩니다.
2. **등급 결정** — 게임 시작 시 랜덤으로 고기 등급이 결정됩니다.
3. **돼지 잡기** — 좌클릭으로 돼지를 점프시키며, 울타리를 피해 제한 시간 동안 생존합니다.
4. **포획 성공** — 체력을 전부 깎으면 성공! 해당 등급의 고기를 보상으로 받습니다.

{% hint style="warning" %}
울타리에 부딪히거나 화면 밖으로 벗어나면 **포획 실패**입니다.
{% endhint %}

## 고기 등급 및 종류

등급이 높을수록 생존 시간이 길어 난이도가 올라가지만, 더 비싼 고기를 얻을 수 있습니다.\
등급은 게임 시작 시 가중치 랜덤으로 결정되며, S·A급은 매우 희귀합니다.
