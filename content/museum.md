# 📖 도감

도감은 네트워크 전체의 수집/기록 시스템입니다.\
낚시·도축 등으로 얻은 아이템이 자동으로 도감에 기록되며, 일정 수집량을 달성하면 보상을 받을 수 있습니다.

## 열기

```
/도감
/도감 낚시
/도감 도축
```

`/도감` 명령어로 메인 메뉴를 열거나, 카테고리 이름을 붙여 바로 진입할 수 있습니다. 별칭: `/박물관`, `/museum`.

## 작동 방식

* 도감의 각 카테고리는 **직접 획득(direct-only)** 방식입니다. 거래·상점·창고 이동 등으로 들어온 아이템은 기록되지 않고, **본인이 직접 낚거나 도축한 경우에만** 카운트됩니다.
* 같은 아이템이라도 등급(LEGEND/EPIC/RARE/COMMON)별로 요구 수량이 다릅니다.
* 모든 진행도는 네트워크 단위로 공유되어 어느 서버에서 잡아도 한 데에 누적됩니다.

## 카테고리

현재 운영 중인 카테고리는 두 가지입니다.

| 카테고리       | 진입 명령어     | 기록 트리거         |
| ---------- | ---------- | -------------- |
| 🎣 낚시 도감 | `/도감 낚시` | 낚시 성공 시 |
| 🐷 도축 도감 | `/도감 도축` | 도축 성공 시 |

## 🎣 낚시 도감

### S급 (LEGEND)

| 이름 | 설명 |
| --- | --- |
| 귀상어 | 망치처럼 넓은 머리로 바다를 누빈다. |
| 청새치 | 날카로운 부리로 파도를 가르는 대형 어종. |
| 대왕가오리 | 거대한 날개로 유유히 헤엄치는 심해의 왕. |

### A급 (EPIC)

| 이름 | 설명 |
| --- | --- |
| 가오리 | 넓은 가슴지느러미로 수면을 미끄러진다. |
| 해파리 | 부드럽고 투명한 실루엣. |
| 랍스터 | 바다의 왕위 도전자. |

| 연어 | 거슬러 오르는 붉은 등. |

### B급 (RARE)

베스, 잉어, 해마, 열대어, 쭈꾸미, 개불, 금붕어, 멸치, 개복치, 불가사리, 메기

### C급 (COMMON)

빈 깡통, 독개구리, 나무 판자, 청개구리, 송사리, 쓰레기 봉투, 캔콜라

| --- | --- |
