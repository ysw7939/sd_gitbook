# 농장 권한

## 권한이란?

농장 권한은 **각 직급별로 할 수 있는 행동을 제한하는 시스템**입니다.\
농장주는 `/농장 권한` 명령어를 통해 멤버, 관리자, 부농장주 등의 직급이 어떤 행동을 할 수 있는지 설정할 수 있습니다.

## 권한 설정 방법

1. `/농장 권한` 명령어를 입력합니다.
2. 권한 GUI가 열리면 변경하고 싶은 항목을 클릭합니다.
3. 클릭할 때마다 최소 직급이 순환됩니다. (농장주 → 부농장주 → 관리자 → 멤버 → 농장주 ...)
4. GUI를 닫으면 변경 사항이 자동으로 저장됩니다.

{% hint style="info" %}
권한을 변경하려면 **멤버 권한 변경** 권한이 필요합니다. 기본적으로 부농장주 이상만 가능합니다.
{% endhint %}

## 권한 목록

### 블록 관련

| 권한 | 설명 | 기본 직급 |
| --- | --- | --- |
| 블록 파괴 | 농장에서 블록을 부술 수 있습니다. | 알바 |
| 블록 설치 | 농장에서 블록을 놓을 수 있습니다. | 알바 |
| 블록 조작 | 문, 버튼, 작업대 등을 사용할 수 있습니다. | 알바 |
| 기능 블록 | 화로, 훈연기, 용광로, 호퍼 등을 사용할 수 있습니다. | 알바 |
| 상자 열기 | 상자, 배럴, 셜커 박스 등을 열 수 있습니다. | 멤버 |
| 액자 사용 | 액자에 아이템을 넣거나 뺄 수 있습니다. | 알바 |

| 권한 | 설명 | 기본 직급 |
| --- | --- | --- |

| 아이템 줍기 | 농장에서 떨어진 아이템을 주울 수 있습니다. | 알바 |

| 권한 | 설명 | 기본 직급 |
| --- | --- | --- |
| 은행 입금 | 농장 은행에 돈을 입금할 수 있습니다. | 멤버 |
| 은행 출금 | 농장 은행에서 돈을 출금할 수 있습니다. | 관리자 |

### 멤버 관리

| 권한 | 설명 | 기본 직급 |
| --- | --- | --- |
| 멤버 초대 | 농장에 새 멤버를 초대할 수 있습니다. | 관리자 |
| 멤버 추방 | 농장에서 멤버를 추방할 수 있습니다. | 관리자 |
| 멤버 권한 변경 | 이 권한 GUI를 열어 권한을 변경할 수 있습니다. | 부농장주 |

### 농장 관리

| 권한 | 설명 | 기본 직급 |
| --- | --- | --- |
| PVP 토글 | 농장의 PVP 설정을 변경할 수 있습니다. | 농장주 |
| 알바 초대 | 농장에 알바를 고용할 수 있습니다. | 부농장주 |
| 알바 해고 | 농장에서 알바를 해고할 수 있습니다. | 부농장주 |
| 월드 시간/날씨 | 농장의 시간, 날씨 설정을 변경할 수 있습니다. | 부농장주 |
| 농장 워프 | 농장 워프를 생성하고 관리할 수 있습니다. | 관리자 |
| 유저 내쫒기 | 농장에 있는 비소속 유저를 내보낼 수 있습니다. | 관리자 |

## 직급 안내

| 직급 | 설명 |
| --- | --- |
| 농장주 | 농장의 소유자입니다. 모든 권한을 가집니다. |
| 부농장주 | 농장주와 유사한 관리 권한을 가집니다. |
| 관리자 | 멤버 초대, 추방 등 일부 관리 권한을 가집니다. |
| 멤버 | 농장의 일반 소속원입니다. |
| 알바 | 임시 고용된 작업자입니다. 농장 소속이 아닙니다. |
| 방문자 | 농장에 방문한 비소속 유저입니다. |

## 멤버 직급 변경

1. `/농장 멤버` 명령어로 멤버 목록 GUI를 엽니다.
2. 직급을 변경할 멤버의 머리를 클릭합니다.
3. 클릭할 때마다 직급이 순환됩니다. (멤버 → 관리자 → 부농장주 → 멤버 ...)
4. GUI를 닫으면 변경 사항이 저장되고 즉시 적용됩니다.

{% hint style="warning" %}
직급 변경은 **농장주만** 할 수 있습니다.
{% endhint %}
