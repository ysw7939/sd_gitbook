# 농장 업그레이드

농장 메뉴(`/농장 업그레이드`)에서 다양한 업그레이드를 진행할 수 있습니다.\
업그레이드 비용은 농장 소유자의 개인 소지금에서 차감됩니다.

{% hint style="warning" %}
업그레이드는 **농장주만** 할 수 있습니다.
{% endhint %}
