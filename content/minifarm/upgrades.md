# 농장 업그레이드

농장 메뉴(`/농장 업그레이드`)에서 다양한 업그레이드를 진행할 수 있습니다.\
업그레이드 비용은 농장 소유자의 개인 소지금에서 차감됩니다.

{% hint style="warning" %}
업그레이드는 **농장주만** 할 수 있습니다.
{% endhint %}

{% tabs %}
{% tab title="크기" %}
농장의 경계 크기를 확장합니다. 표시된 값은 농장 중심을 기준으로 한 **가로×세로 면적**입니다.

| 레벨 | 크기 (가로×세로) | 비용         |
| -- | ----------- | ---------- |
| 기본 | 51×51       | 무료         |
| 1  | 101×101     | 30만원       |
| 2  | 201×201     | 150만원      |
| 3  | 301×301     | 800만원      |
| 4  | 501×501     | 5,000만원    |
| 5  | 701×701     | 1억 1,500만원 |
| 6  | 1001×1001   | 3억원        |
{% endtab %}
