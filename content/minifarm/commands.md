# 농장 명령어

마인팜에서 사용할 수 있는 명령어 목록입니다.

{% hint style="info" %}
명령어는 업데이트에 따라 추가/변경될 수 있습니다.
{% endhint %}

{% tabs %}
{% tab title="기본" %}
| 명령어             | 단축키  | 설명             |
| --------------- | ---- | -------------- |
| `/농장`           | `/농` | 농장 관리 메뉴 열기    |
| `/농장 생성 <이름>`   |      | 새 농장 생성        |
| `/농장 이동`        |      | 내 농장으로 이동      |
| `/농장 삭제`        |      | 농장 삭제 (확인 GUI) |
| `/농장 이름변경 <이름>` |      | 농장 이름 변경       |
| `/농장 양도 <닉네임>`  |      | 농장장 권한 양도      |
| `/농장 탈퇴`        |      | 소속된 농장에서 탈퇴    |
| `/농장 순위`        |      | 농장 순위 보기       |
{% endtab %}

{% tab title="멤버" %}
| 명령어              | 단축키 | 설명             |
| ---------------- | --- | -------------- |

| `/농장 초대 <닉네임>`   |     | 다른 유저를 농장에 초대  |
| `/농장 수락 <닉네임>`   |     | 받은 농장 초대 수락    |
| `/농장 추방 <닉네임>`   |     | 농장 멤버 추방       |
| `/농장 권한`         |     | 멤버 등급별 권한 설정   |
| `/농장 알바고용 <닉네임>` |     | 알바(코업) 고용      |
| `/농장 알바해고 <닉네임>` |     | 알바 해고          |
| `/농장 알바이동`       |     | 내가 알바인 농장으로 이동 |
{% endtab %}

{% tab title="방문 / 워프" %}

| `/농장 워프생성 <이름>`  |     | 현재 위치를 워프로 등록 |

| `/농장 워프삭제 <이름>`  |     | 워프 삭제         |
{% endtab %}
