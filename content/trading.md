# 🚢 무역

무역은 아이템을 일반 상점보다 **높은 가격**에 판매할 수 있는 시스템입니다.\
무역상인 **루루**를 우클릭하거나, `/원격무역`(대장 등급) 명령어로 이용할 수 있습니다.

## 시스템 안내

* 무역 가격은 일반 판매가 대비 **최대 20%** 높게 판매 가능합니다.
* 매일 **오전 3시**에 일일 판매량이 초기화됩니다.
* 빈 슬롯에 아이템을 등록하고, 가격을 조정하여 판매합니다.

## 무역 슬롯

무역 슬롯은 [등급](../ranks.md)에 따라 해금됩니다.

| 등급 | 슬롯 수 |
| --- | --- |
| 아기 | 1칸 |
| 꼬마 | 2칸 |
| 의젓 | 3칸 |
| 듬직 | 4칸 |
| 대장 | 5칸 |

## 무역 등급

무역 등급이 높을수록 일일 판매량과 프리미엄이 증가합니다.

| 등급 | 일일 판매 한도 (슬롯당) | 최대 프리미엄 |
| --- | --- | --- |
| 기본 | 64개 | +5% |
| 레어 | 128개 | +10% |
| 엘리트 | 192개 | +15% |
| 히어로 | 256개 | +18% |
| 레전드 | 320개 | +20% |

{% hint style="info" %}
무역 등급은 무역 활동을 통해 승급할 수 있습니다.
{% endhint %}
