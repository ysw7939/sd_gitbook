# 🫧 진주 줍기

진주는 농사 또는 채굴 중 확률적으로 획득할 수 있는 특별 아이템입니다.

<figure><img src="../.gitbook/assets/image (17).png" alt=""><figcaption></figcaption></figure>

## 진주 드롭 확률

### 농장 서버

| 블록           | 확률   |
| ------------ | ---- |
| 밀, 호박, 수박    | 1.2% |
| 당근, 감자       | 1.4% |
| 비트            | 1.5% |
| 참나무/짙은 참나무 잎 | 1.1% |

### 광산 서버

| 블록       | 확률   |
| -------- | ---- |
| 조약돌, 돌   | 0.1% |
| 석탄 광석    | 0.8% |
| 철 광석     | 0.8% |
| 금 광석     | 1.1% |
| 레드스톤 광석  | 1.1% |
| 다이아몬드 광석 | 3.0% |
| 에메랄드 광석  | 3.0% |
| 고대 잔해    | 5.0% |

{% hint style="info" %}
심층암 변종(예: 심층암 철 광석)도 동일한 확률이 적용됩니다.
{% endhint %}

## 진주 성장

수집한 진주는 `/진주 주기` 명령어로 수달에게 줄 수 있습니다.\
진주를 줄 때마다 **10포인트**씩 적립되며, 포인트가 쌓이면 진주가 성장합니다.

| 성장 단계 | 필요 포인트 |
| ----- | ------ |
| 1단계   | 50     |
| 2단계   | 150    |
| 3단계   | 300    |

## 진주 줍기 단계 및 보상

| 단계     | 필요 진주    | 보상                                                         |
| ------ | -------- | ---------------------------------------------------------- |
| 초보 조개꾼 | 0 (기본단계) | 없음                                                         |
| 숙련 수집가 | 500      | <p>500 캐시<br>플라이 쿠폰(30분) 1개<br>중첩 티켓 32개<br>광물 스크롤 10개</p> |
| 진주 감별사 | 2,500    | <p>1,000 캐시<br>플라이 쿠폰(30분) 2개<br>중첩티켓 32개</p>              |
| 진주 마스터 | 7,000    | <p>2,000 캐시<br>맑은 물 10개<br>중첩 티켓 256개<br>광물 스크롤 20개</p>    |
| 대왕 진주  | 25,000   | <p>[대왕 진주] 칭호<br>3,000 캐시<br>쿨 식스 6개<br>중첩티켓 512개</p>      |

## 시즌 랭킹

진주 포인트에 따라 시즌 랭킹(상위 10명)이 집계됩니다.\
랭킹은 `/진주 랭킹` 명령어로 확인할 수 있습니다.

{% hint style="info" %}
순위는 매월 25일 자정에 집계됩니다.

신속함, 성급함 효과 지속시간은 다음 시즌까지입니다.
{% endhint %}
