# 🫧 진주 줍기

진주는 농사 또는 채굴 중 확률적으로 획득할 수 있는 특별 아이템입니다.

<figure><img src="../.gitbook/assets/image (18).png" alt=""><figcaption></figcaption></figure>

## 진주 드롭 확률

### 농장 서버

| 블록 | 확률 |
| --- | --- |
| 밀, 호박, 수박 | 1.2% |
| 당근, 감자 | 1.4% |
| 참나무/짙은 참나무 잎 | 1.1% |

### 광산 서버

| 블록 | 확률 |
| --- | --- |
| 조약돌, 돌 | 0.1% |
| 석탄 광석 | 0.8% |
| 철 광석 | 0.8% |
| 금 광석 | 1.1% |
| 레드스톤 광석 | 1.1% |
| 다이아몬드 광석 | 3.0% |
| 에메랄드 광석 | 3.0% |
| 고대 잔해 | 5.0% |

{% hint style="info" %}
심층암 변종(예: 심층암 철 광석)도 동일한 확률이 적용됩니다.
{% endhint %}

## 진주 성장

수집한 진주는 `/진주 주기` 명령어로 수달에게 줄 수 있습니다.\
진주를 줄 때마다 **10포인트**씩 적립되며, 포인트가 쌓이면 진주가 성장합니다.

| 성장 단계 | 필요 포인트 |
| --- | --- |
| 1단계 | 50 |
| 2단계 | 150 |
| 3단계 | 300 |

## 시즌 랭킹

진주 포인트에 따라 시즌 랭킹(상위 10명)이 집계됩니다.\
랭킹은 `/진주 랭킹` 명령어로 확인할 수 있습니다.
