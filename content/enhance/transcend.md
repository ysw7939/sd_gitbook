# 🔥 초월 강화

초월 강화는 장비의 **등급(티어)**&#xC744; 올려 특수 효과를 부여하는 강화입니다.

<figure><img src="../../.gitbook/assets/image (7).png" alt=""><figcaption></figcaption></figure>

## 강화 방법

1. 강화 메뉴를 열고 장비를 등록합니다.
2. **초월 스크롤**을 재료로 사용합니다.
3. 골드를 소모하여 강화를 시도합니다.

## 곡괭이 초월 등급

| 등급 | 필요 레벨 | 재료           | 비용         | 성공 확률 | 하락 위험 | 추가 광물 확률 | 진주 확률 |
| -- | ----- | ------------ | ---------- | ----- | ----- | -------- | ----- |
| 1  | 5     | 초월 스크롤 (I)   | 100,000원   | 60%   | 없음    | +5%      | +1.0% |
| 2  | 10    | 초월 스크롤 (II)  | 250,000원   | 60%   | 없음    | +10%     | +1.5% |
| 3  | 15    | 초월 스크롤 (III) | 500,000원   | 50%   | 없음    | +15%     | +2.0% |
| 4  | 20    | 초월 스크롤 (IV)  | 1,000,000원 | 45%   | 10%   | +20%     | +2.5% |
| 5  | 25    | 초월 스크롤 (V)   | 2,000,000원 | 35%   | 10%   | +20%     | +3.0% |

{% hint style="warning" %}
**4등급 이상**부터 실패 시 등급이 **하락**할 수 있습니다!
{% endhint %}

{% hint style="success" %}
**5등급** 달성 시 특수 효과: 캔 광물이 자동으로 제련되는 **태양열 효과**가 적용됩니다!
{% endhint %}

## 괭이 초월 등급

괭이 초월은 레벨 조건 없이 골드와 초월 스크롤만으로 강화할 수 있습니다.

| 등급 | 재료           | 비용         | 성공 확률 | 하락 위험 |
| -- | ------------ | ---------- | ----- | ----- |
| 1  | 초월 스크롤 (I)   | 100,000원   | 60%   | 없음    |
| 2  | 초월 스크롤 (II)  | 250,000원   | 60%   | 없음    |
| 3  | 초월 스크롤 (III) | 500,000원   | 50%   | 없음    |
| 4  | 초월 스크롤 (IV)  | 1,000,000원 | 45%   | 10%   |
| 5  | 초월 스크롤 (V)   | 2,000,000원 | 35%   | 10%   |

## 보호 스크롤

보호 스크롤을 함께 사용하면 실패 시 등급 하락을 막아줄 수 있습니다.

| 보호 스크롤     | 보호 성공 확률 |
| ---------- | -------- |
| 보호 주문서 I   | 60%      |
| 보호 주문서 II  | 70%      |
| 보호 주문서 III | 80%      |
