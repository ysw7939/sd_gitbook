# 📖 인첸트 강화

인첸트 강화는 도구의 기본 능력치(효율, 내구, 행운)를 올려주는 강화입니다.\
재료 슬롯에 **수달석** 또는 **윤기나는 수달석**을 넣으면 자동으로 인첸트 강화 모드가 됩니다.

<figure><img src="../../.gitbook/assets/image (18).png" alt=""><figcaption></figcaption></figure>

## 강화 방법

1. `/메뉴` → **강화** → NPC **베리** 우클릭으로 강화 GUI를 엽니다.
2. **재료 슬롯**에 수달석 또는 윤기나는 수달석을 넣습니다.
3. **도구 슬롯**에 강화할 도구를 넣습니다.
4. 강화 버튼을 클릭하여 시도합니다.

## 재료별 성공 확률

어떤 등급이든 재료는 자유롭게 선택할 수 있으며, 재료에 따라 성공 확률이 달라집니다.

| 재료       | 성공 확률 |
| -------- | ----- |
| 수달석      | 45%   |
| 윤기나는 수달석 | 65%   |

{% hint style="info" %}
비싼 재료일수록 성공 확률이 높습니다. 등급 제한 없이 원하는 재료를 선택할 수 있습니다.
{% endhint %}
