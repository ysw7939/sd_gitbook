# ⛰️ 광산

광산은 메인 월드와 분리된 별도의 채굴 전용 월드입니다.\
`Shift + F` 메뉴에서 워프를 통해 광산으로 이동할 수 있습니다.

## 특징

* 수달농장만의 광물 분포를 유지합니다.
* 돌, 흙 등의 블록은 광산 월드에서만 얻을 수 있습니다.
* 채굴 스탯과 곡괭이 강화를 활용하여 효율적으로 채굴할 수 있습니다.
* 광산 월드는 **매일 오전 3시**에 초기화됩니다.

{% hint style="warning" %}
초기화 시 광산에 있는 플레이어는 자동으로 스폰으로 이동됩니다.
{% endhint %}

## 특수 드롭

채굴 중 확률적으로 특수 아이템이 드롭될 수 있습니다.

| 아이템        | 기본 드롭 확률 | 용도                        |
| ---------- | -------- | ------------------------- |
| 수달석        | 1%       | 인첸트 강화 재료 (성공률 45%)       |
| 윤기나는 수달석   | 0.024%   | 인첸트 강화 재료 (성공률 65%)       |
| 섬세한 손길 주문서 | 0.005%   | 도구에 사용 시 **섬세한 손길** 효과 부여 |
| 네더라이트 형판   | 0.05%    | 네더라이트 장비 제작 재료           |

{% hint style="info" %}
특수 드롭은 아이템별로 각각 확률이 계산됩니다.\
실제 확률은 `기본 드롭 확률 × (1 + 곡괭이 커스텀 드롭 보너스 + 인첸트 강화 레벨 보너스)`로 적용됩니다.\
수달석, 섬세한 손길 주문서, 네더라이트 형판은 기본 곡괭이부터 드롭될 수 있으며, 윤기나는 수달석은 수달 곡괭이 I 이상부터 드롭될 수 있습니다.
{% endhint %}

### 섬세한 손길 주문서 사용법

1. 인벤토리에서 **섬세한 손길 주문서**를 커서로 집습니다.
2. 곡괭이 / 도끼 / 삽 / 괭이 위에 올려서 좌클릭하면 즉시 적용됩니다.
3. 적용 후 도구 lore 맨 아래에 `섬세한 손길 +1`이 표시됩니다.

{% hint style="success" %}
섬세한 손길은 **인첸트 강화 실패로 초기화돼도 보존**됩니다.\
또한 초월 강화로 새 등급의 곡괭이로 교체되어도 그대로 유지됩니다.
{% endhint %}

{% hint style="warning" %}
이미 섬세한 손길이 부여된 도구에는 다시 사용할 수 없습니다.
{% endhint %}
