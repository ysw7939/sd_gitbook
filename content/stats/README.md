# 📜 스탯

수달농장에는 **채굴 스탯**과 **농사 스탯** 두 가지 스킬 시스템이 있습니다.

* 채굴 스탯: `/스탯` 또는 `/ㅅㅌ`
* 농사 스탯: `/채집스탯`

## 공통 규칙

* 모든 스킬은 **최대 Lv.5**까지 올릴 수 있습니다.
* 스킬 레벨업에는 **스킬 포인트**와 **골드**가 필요합니다.
* 스킬 포인트는 레벨업 시 획득합니다.
  * 채굴: 레벨업당 **+5 포인트**
  * 농사: 레벨업당 **+1 포인트**

## 스킬 레벨업 비용

| 스킬 레벨 | 필요 포인트 | 필요 골드 |
| --- | --- | --- |
| Lv.1 | 1 | 5,000원 |
| Lv.2 | 2 | 12,000원 |
| Lv.3 | 5 | 35,000원 |
| Lv.4 | 12 | 120,000원 |
| Lv.5 | 25 | 400,000원 |
