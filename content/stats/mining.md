# ⛏️ 채굴 스킬

채굴 스탯은 광물을 캐면서 경험치를 얻고, 스킬을 배워 채굴 효율을 높이는 시스템입니다.\
`/스탯` 또는 `/ㅅㅌ` 명령어로 확인할 수 있습니다.

## 곡괭이 등급별 경험치 보너스

곡괭이의 초월 등급이 높을수록 광물당 획득 경험치가 증가합니다.

| 등급 | 광물당 경험치 |
| --- | --- |
| 1등급 | +1 |
| 2등급 | +1 |
| 3등급 | +2 |
| 4등급 | +4 |
| 5등급 | +8 |
| 6등급 | +12 |
| 7등급 | +18 |
| 8등급 | +20 |

## 스킬 목록

### 1. 채굴 숙련

채굴 경험치 획득량이 증가합니다.

| 레벨 | 효과 |
| --- | --- |
| Lv.1 | +8% |
| Lv.2 | +16% |
| Lv.3 | +25% |
| Lv.4 | +35% |
| Lv.5 | +50% |

### 2. 피버타임

채굴 시 확률적으로 피버타임이 발동하여 드롭이 **2배**가 됩니다.

| 레벨 | 발동 확률 | 지속시간 | 추가 효과 |
| --- | --- | --- | --- |
| Lv.1 | 0.04% | 20초 | - |
| Lv.2 | 0.06% | 25초 | - |
| Lv.3 | 0.08% | 35초 | - |
| Lv.4 | 0.10% | 45초 | 신속 I |
| Lv.5 | 0.12% | 60초 | 신속 I + 야간투시 |

### 3. 성급함

채굴 시 확률적으로 성급함 효과가 부여됩니다.

| 레벨 | 발동 확률 | 지속시간 | 효과 | 추가 효과 |
| --- | --- | --- | --- | --- |
| Lv.1 | 0.15% | 8초 | 성급함 I | - |
| Lv.2 | 0.25% | 10초 | 성급함 I | - |
| Lv.3 | 0.35% | 14초 | 성급함 II | - |
| Lv.4 | 0.45% | 18초 | 성급함 II | - |
| Lv.5 | 0.60% | 25초 | 성급함 II | 신속 I |

### 4. 광산왕

{% hint style="warning" %}
**해금 조건**: 채굴 숙련, 피버타임, 성급함 스킬이 모두 **Lv.5**에 도달해야 합니다.
{% endhint %}
