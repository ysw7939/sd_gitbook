# 🏷️ 단축어

자주 사용하는 명령어를 짧은 별칭으로 등록하여 빠르게 사용할 수 있는 기능입니다.

## 사용 방법

| 명령어 | 설명 |
| --- | --- |
| `/단축어 <원본명령어> <별칭>` | 단축어 등록 |
| `/단축어 삭제 <별칭>` | 단축어 삭제 |
| `/단축어 목록` | 등록한 단축어 목록 확인 |

## 예시

* `/단축어 워프 ㅇ` — 이후 `/ㅇ` 입력 시 `/워프`가 실행됩니다.

{% hint style="info" %}
1인당 최대 **5개**의 단축어를 등록할 수 있습니다. (권한에 따라 10개까지 확장 가능)
{% endhint %}
